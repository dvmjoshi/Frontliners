# Frontliners
Project Submission  for 30 Days of Kotlin with Google Developers
## Features
* Covid 19 India Tracker 
* Show Current Status From Api
* List Of Cases Statewise
## Kotlin Concept used
* Kotlin Singleton
* Coroutine
* Data Classes
* late-initialized properties
* Named Parameter
